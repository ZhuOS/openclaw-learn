教程定位

这不是一份「怎么用 OpenClaw」的使用文档 — 官方文档 docs.openclaw.ai 已经够详细。

这是一份面向开发者的源码学习指南，目标是让你：

① 看懂：一个生产级 AI Agent Gateway 是怎么设计出来的
② 读懂：能顺着入口一路追到任意子系统的核心代码
③ 改得动：自己写一个 Tool / Skill / Plugin / Channel 适配器

📦 源码：github.com/openclaw/openclaw 📚 官方文档：docs.openclaw.ai 🦞 作者：@steipete

📚 1 周系统学习

按顺序每天 1 章，配合源码边读边改，每章做笔记。适合想真正吃透这套架构的工程师。

🎯 针对性查阅

各章独立完整，遇到问题直接查对应章节。例如做插件开发直接看 Ch07 + Ch09。

适合人群

✅ Node.js / TypeScript 有一定基础（看得懂 async/await、workspace、ESM）

✅ 对 AI Agent、工具调用（tool calling）、LLM 工程有基本认知

✅ 想搞清楚「消息网关 / 多渠道适配 / 多 Agent 路由」在工业级产品里到底怎么写

✅ 希望自建 AI 助手，把数据和控制权掌握在自己手上
